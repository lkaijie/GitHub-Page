About

QRHunter is an android app for a Pokemon-Go-like game. This app was developed as a group project for CMPUT 301 and it is also my first project involving many people (6!). This is a very memorable project for me as this is where I first got into using github and version control. I also learned a lot about the importance of communication and teamwork. This was a genuinely fun project to work on and I am very proud of the end result, I got to learn how to implement an android app from scratch following the official android architecture guidelines. The main things I learnt from this were working as a team using Github, Test-Driven-Development, Database integration and of course Android development.


Features

The main functionality of this application is to allow users to scan QR codes for points, earning their spot on the prestigious leaderboard.


Main features

  • Scan QR or Bar codes to optain points and earn a visual representation of your scanned QRcode
  • Leaderboard: Display or search for players, the top 3 in the leaderboards earn a special badge
  • Profile: Check your scanned QR codes for their score rankings or your leaderboard position, you can also add comments to QRcodes that you have scanned
  • Map: Uses the google maps API to display any nearby scanned codes
